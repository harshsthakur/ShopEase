# 🍋 ShopEase – Smart Wellness & Food Commerce Platform

A full-stack food-commerce platform focused on premium natural products such as Amla, Mango, Modak, and Fruit Jam.

The platform combines modern e-commerce functionality with wellness-focused shopping experiences, seasonal product promotions, and festival-aware marketing.

---

## 🚀 Project Overview

ShopEase is a modern food-commerce platform designed to provide customers with a personalized shopping experience for healthy and traditional food products.

Unlike traditional e-commerce platforms, ShopEase focuses on:

* Wellness-based product discovery
* Seasonal product promotion
* Festival-specific collections
* Subscription-ready architecture
* Comprehensive admin management

### Unique Features

A dynamic seasonal marketing feature that automatically promotes relevant products based on upcoming festivals and seasonal events.

Examples:

* Ganesh Chaturthi Special Modak Collection
* Mango Season Collection
* Winter Wellness Collection

#### 🎁 Festival Gift Boxes

Curated gift boxes for seasonal celebrations and traditional festivals.

#### 📦 Subscription Box System (Planned)

Monthly wellness and food subscription boxes with recurring deliveries.

Personalized product recommendations based on customer wellness goals and dietary preferences.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/harshthakur/ShopEase.git
```

### Install Dependencies

```bash
cd frontend
npm install

cd ../backend
npm install
```

### Start Backend

```bash
cd backend
npm start
```

### Start Frontend

```bash
cd frontend
npm run dev
```
