# ShopEase - Full Stack E-Commerce Platform

A modern food-commerce platform focused on premium natural products including Amla, Mango, Modak, and Fruit Jam.

## 🚀 Features

### Customer Features

* User Registration & Login
* Product Search & Filtering
* Shopping Cart
* Checkout System
* Order History
* Wishlist
* Festival-Based Product Recommendations

### Admin Features

* Admin Dashboard
* Product Management
* Category Management
* Inventory Management
* Customer Management
* Sales Analytics
* Order Tracking

### Unique Features

* 🎉 Festival Countdown Store
* 🎁 Festival Gift Boxes
* 🥭 Seasonal Product Promotions
* 📦 Subscription Box System
* 🍋 Wellness-Focused Product Collections

## 🛠️ Tech Stack

### Frontend

* React.js
* Vite
* Tailwind CSS
* React Router

### Backend

* Node.js
* Express.js

### Database

* JSON Database Storage
* MongoDB Ready

## 📂 Project Structure

shopease/

## ⚙️ Installation

### Clone Repository

git clone YOUR_REPOSITORY_URL

### Install Dependencies

cd frontend
npm install

cd ../backend
npm install

### Run Backend

cd backend
npm start

### Run Frontend

cd frontend
npm run dev
